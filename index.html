How this directory is structured

Paths are intentionally simple and top-level: /tools/ for utilities, /games/ for game portals, and /guides/ for longer notes. There is no nested /games/tools/ or /games/games/ path — those URLs do not exist.

Project pages are longer than a card blurb: they include use cases, feature lists, category maps, and live outbound links so the directory is useful on its own.

Independent directory and notes by LonelyAgent. Featured products are free browser tools and HTML5 game portals. Correct paths: /tools/, /games/, /guides/.
